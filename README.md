# SalatTrack

## Project Overview
SalatTrack is a comprehensive application designed to help users track their daily prayers and manage their spiritual journey effectively. It provides a user-friendly interface that promotes regular prayer habits.

## Features
- **Prayer Time Alerts**: Users receive notifications for prayer times based on their location.
- **Qibla Direction**: Accurate guidance towards Qibla for users worldwide.
- **Prayer History**: Record and view past prayer metrics to track consistency.
- **Community Features**: Connect with friends and share progress.
- **Customization Options**: Personalize alerts and interface settings.

## Architecture
The SalatTrack application follows a microservices architecture, allowing for scalability and easier maintenance. Key components include:
- **Frontend**: A responsive web interface built with React.
- **Backend**: An API built with Node.js and Express, interacting with a MongoDB database.
- **Notification Service**: A dedicated service for managing alerts and notifications.

## Tech Stack
- **Frontend**: React, Redux, CSS, HTML
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Docker, AWS
- **Testing**: Jest, Mocha

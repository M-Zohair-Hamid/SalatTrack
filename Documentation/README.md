# SalatTrack

This is the main documentation folder for SalatTrack Project.

## Project Structure

- **Presentation**: Contains all the UI components.
- **BusinessLogic**: Contains the core business logic of the application.
- **DataAccess**: Manages data persistence and database interactions.
- **Tests**: Includes all unit and integration tests.
- **Documentation**: Holds project documentation and references.
- **Resources**: Contains images, icons, and other necessary resources.
